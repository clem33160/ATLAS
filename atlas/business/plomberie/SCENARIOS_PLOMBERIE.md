# SCENARIOS PLOMBERIE

## 1) Fuite urgente
- Symptômes : eau qui coule en continu, augmentation rapide des dégâts.
- Niveau d’urgence : critique.
- Questions à poser : origine visible ? robinet d’arrêt accessible ?
- Réponse client préparée : consignes de sécurité + proposition d’intervention prioritaire.
- Risque : aggravation dégât des eaux.
- Prochaine action humaine : appel immédiat opérateur.

## 2) WC bouché
- Symptômes : remontées, évacuation impossible.
- Niveau d’urgence : élevé.
- Questions à poser : logement unique avec un seul WC ?
- Réponse client préparée : qualification rapide + créneau intervention.
- Risque : débordement sanitaire.
- Prochaine action humaine : validation intervention rapide.

## 3) Évier bouché
- Symptômes : écoulement très lent, reflux.
- Niveau d’urgence : moyen.
- Questions à poser : depuis quand ? cuisine inutilisable ?
- Réponse client préparée : proposition débouchage et fourchette indicative.
- Risque : aggravation obstruction.
- Prochaine action humaine : priorisation planning.

## 4) Chauffe-eau en panne
- Symptômes : absence d’eau chaude.
- Niveau d’urgence : moyen à élevé.
- Questions à poser : âge équipement ? disjonction ? fuite associée ?
- Réponse client préparée : diagnostic sur site recommandé.
- Risque : confusion panne simple/risque électrique.
- Prochaine action humaine : technicien qualifié.

## 5) Chaudière en panne
- Symptômes : chauffage et/ou eau chaude indisponibles.
- Niveau d’urgence : élevé en hiver.
- Questions à poser : code erreur, odeur inhabituelle ?
- Réponse client préparée : assistance prioritaire + sécurité.
- Risque : risque sécurité si anomalie combustion.
- Prochaine action humaine : validation expert chaudière.

## 6) Dégât des eaux
- Symptômes : traces d’humidité, plafond/murs touchés.
- Niveau d’urgence : élevé.
- Questions à poser : source active ? voisins impactés ?
- Réponse client préparée : stopper l’eau + dossier assurance.
- Risque : extension des dommages.
- Prochaine action humaine : coordination intervention + assurance.

## 7) Remplacement robinetterie
- Symptômes : fuite légère, robinet vétuste.
- Niveau d’urgence : faible à moyen.
- Questions à poser : modèle souhaité ? urgence esthétique/fonctionnelle ?
- Réponse client préparée : devis planifié.
- Risque : incompatibilité matériel.
- Prochaine action humaine : validation devis.

## 8) Rénovation salle de bain
- Symptômes : projet planifié, modernisation complète.
- Niveau d’urgence : faible.
- Questions à poser : budget, délais, périmètre travaux ?
- Réponse client préparée : cadrage besoin + visite technique.
- Risque : dérive coûts/délais.
- Prochaine action humaine : chef de projet travaux.

## 9) Canalisation bouchée
- Symptômes : plusieurs évacuations lentes ou bloquées.
- Niveau d’urgence : élevé.
- Questions à poser : maison/appartement ? historique incidents ?
- Réponse client préparée : intervention ciblée inspection/débouchage.
- Risque : obstruction réseau principal.
- Prochaine action humaine : technicien équipé.

## 10) Demande de devis non urgente
- Symptômes : besoin d’estimation sans panne immédiate.
- Niveau d’urgence : faible.
- Questions à poser : nature travaux, délai souhaité, localisation.
- Réponse client préparée : parcours devis structuré.
- Risque : perte du lead faute de relance.
- Prochaine action humaine : relance commerciale.
