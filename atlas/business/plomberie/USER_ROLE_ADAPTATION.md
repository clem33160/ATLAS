# USER ROLE ADAPTATION

Rôles supportés: client inquiet, artisan pressé, technicien, responsable planning, gérant, assistant administratif, syndic, propriétaire, locataire, client professionnel.

Pour chaque rôle:
- niveau de détail
- niveau de prudence
- sortie prioritaire

Exemple artisan pressé: réponse courte + action immédiate + dispatch.
Exemple client inquiet: rassurer + étapes + sécurité.
