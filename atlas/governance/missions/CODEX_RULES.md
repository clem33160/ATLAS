# CODEX RULES

- Codex ne décide pas seul du canon.
- Codex implémente dans les chemins autorisés.
- Codex doit lancer les tests.
- Codex doit produire un rapport.
- Codex doit signaler les incertitudes.
- Codex doit refuser une mission impossible ou dangereuse.
- Codex doit éviter les changements globaux non demandés.
- Codex doit respecter AGENTS.md.
