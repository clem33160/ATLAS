# MISSION CONTRACT

Toute mission Codex doit contenir :
- Mission ID
- Objectif
- Domaine
- Type de mission
- Contexte
- Chemins autorisés
- Chemins interdits
- Fichiers à ne jamais toucher
- Règles obligatoires
- Tests obligatoires
- Critères d’acceptation
- Critères de refus
- Rapport final attendu
- Niveau de validation humaine requis
