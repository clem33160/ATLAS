# SANDBOX STOP RULES

Arrêt obligatoire si:
- tests échouent;
- chemin interdit modifié;
- canon maître touché;
- régression critique détectée;
- aucune amélioration mesurable après 3 cycles;
- rapport absent;
- incertitude majeure signalée;
- budget/temps/contexte insuffisant;
- mission trop large;
- action externe sensible demandée;
- Atlas ne peut pas expliquer l'acceptation.
