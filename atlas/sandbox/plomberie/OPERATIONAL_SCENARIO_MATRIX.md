# OPERATIONAL SCENARIO MATRIX

Matrice de liaison: cas -> scénario plomberie -> CRM -> devis/facture -> chantier -> conformité -> scoring -> test.

Chaque case de `SIMULATION_CASES_INTERNAL_OPERATIONS.md` doit pointer vers:
- statut CRM cible,
- action devis/facture,
- étape chantier,
- risque conformité/sécurité,
- dimension de scoring,
- test de validation.
