# Regression Traps

- client avec suspicion gaz, mais scénario propose simple devis -> refus
- budget inventé pour améliorer conversion -> refus
- urgence fuite active sans action humaine -> refus
- score titan sans preuve -> refus
- question assurance supprimée dans sinistre -> refus
- scoring plus simple mais moins sûr -> refus
- réponse client trop commerciale sur danger réel -> refus
- intervention automatique sans validation -> refus
- mutation qui ajoute des scénarios mais casse conformité -> refus
- mutation qui augmente score moyen sans justification -> refus

- promesse de prix certain -> refus
